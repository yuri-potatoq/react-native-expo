# react-native-expo

A simple playground where i put some resources about what i have been learn at react-native framework.

### Set up environment

```bash
$ npm i --global expo-cli 
$ expo init <project-name>
$ npm i # at project folder
```

*With flake nix*
```bash
$ nix develop && npm i
```

<tr>

*Running application emulator* 
```bash
$ expo start 
# or
$ npm start
```
